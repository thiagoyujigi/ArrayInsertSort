Método de Ordenação Insert Sort


Funcionamento:

É um algoritmo de ordenação

Constrói uma matriz final com um elemento por vez.

É bastante eficiente para problemas com pequenas entradas.

Percorre as posições do array, começando com o índice 1(um).

Á medida que avança vai deixando os elementos mais à esquerda ordenados.


Representação Gráfica Animação Vídeo

Vantagens:

  • É para ser utilizado quando o arquivo está quase ordenado.
  • Um bom método para adicionar poucos elementos em um arquivo já ordenado.
  • É um algoritmo de ordenação estável.
  • É de simples implementação, leitura e manutenção.
  • É de ágil mobilidade.

Desvantagens:

  • Custo elevado de movimentação de elementos no vetor.
